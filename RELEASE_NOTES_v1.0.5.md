# v1.0.5 - OCR精度向上とページめくり方向設定追加

## 🎯 主な改善点

### 1. OCR精度の大幅向上
- **2倍スケーリング**: 小さな文字の認識精度が向上しました（従来の1.5倍から2倍に増加）
- **シャープネスフィルター**: エッジ強調により文字の輪郭がより明瞭になりました
- **適応的コントラスト調整**: ヒストグラム分析による自動コントラスト最適化で、低コントラスト画像でも高精度に認識できます

### 2. ページめくり方向の設定機能 ✨ NEW
- **右タップで次ページ（通常モード）**: 通常の電子書籍の読み方に対応
- **左タップで次ページ（漫画モード）**: 漫画や右綴じの本に対応
- ユーザーが自由に選択可能な設定UI追加

### 3. ページめくり後の読み上げ開始修正
- **リトライロジック実装**: 最大3回の自動リトライでページめくり後の読み上げ失敗を防止
- **初期遅延追加**: 1.5秒の待機時間で画面安定化を確認
- **テキスト認識確認**: OCR結果の検証機能で確実に読み上げを開始

## 📦 インストール方法

1. 下記の `app-release.apk` をダウンロード
2. Android端末にインストール
3. 必要な権限を許可してご利用ください

## 🔧 必要な権限

- **画面キャプチャ権限**: Kindleの画面を読み取るために必要
- **アクセシビリティ権限**: 自動ページめくり機能に必要
- **オーバーレイ表示権限**: 読み上げコントロールUIの表示に必要

## 🐛 修正された問題

- ページめくり後に読み上げが停止する問題
- ページめくり方向が一定しない問題
- OCR認識精度が低い問題

## 📝 変更されたファイル

- `app/build.gradle` - バージョン更新（1.0.5）
- `app/src/main/java/com/kindletts/reader/OverlayService.kt` - OCR前処理強化、リトライロジック追加
- `app/src/main/java/com/kindletts/reader/MainActivity.kt` - ページ方向設定UI追加
- `app/src/main/res/layout/activity_main.xml` - RadioGroup UI追加
